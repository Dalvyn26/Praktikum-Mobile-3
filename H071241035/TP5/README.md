# Tugas Praktikum 5 - SharedPreferences

## Deskripsi Proyek
**MyBrew Preferences** TP5 adalah aplikasi Android Native yang mendemonstrasikan implementasi sistem penyimpanan lokal menggunakan **SharedPreferences**. Proyek ini mencakup operasi CRUD lengkap, manajemen sesi autentikasi berbasis NIM, penyimpanan media dalam format Base64, serta pengaturan tema aplikasi (Dark/Light Mode) yang persisten.
