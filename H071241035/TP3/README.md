# Tugas Praktikum 3 - Library App | Fragments
## Deskripsi Proyek
**Library App** adalah aplikasi Android yang memungkinkan pengguna untuk mengelola katalog buku secara personal. Proyek ini merupakan pemenuhan **Tugas Praktikum 3 - Mobile Programming**, yang berfokus pada penggunaan `Fragment`, `BottomNavigationView`, `RecyclerView` dengan fitur pencarian dan filter genre, serta pengiriman data objek menggunakan `Parcelable`.
