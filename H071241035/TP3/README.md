# Tugas Praktikum 3 & 4 - Library App

## Update Tugas Praktikum 4: Background Thread & Responsivitas
Pada update kali ini (TP4), aplikasi telah ditingkatkan untuk mendukung proses **Background Threading** demi memastikan antarmuka (UI) tetap responsif saat melakukan pengolahan data berat.

### Fitur Baru di TP4:
- **Background Threading**: Implementasi `ExecutorService` dan `Handler` untuk proses pencarian buku dan pemuatan menu favorit.
- **Progress Bar**: Animasi loading yang muncul selama proses data di latar belakang berjalan.
- **Optimasi Search**: Pencarian kini tidak lagi membebani thread utama.

## Tugas Praktikum 3 - Library App | Fragments
### Deskripsi Proyek
**Library App** adalah aplikasi Android yang memungkinkan pengguna untuk mengelola katalog buku secara personal. Proyek ini merupakan pemenuhan **Tugas Praktikum 3 - Mobile Programming**, yang berfokus pada penggunaan `Fragment`, `BottomNavigationView`, `RecyclerView` dengan fitur pencarian dan filter genre, serta pengiriman data objek menggunakan `Parcelable`.
